1. What is Semantic HTML?

2. What is HTML used for?

3. What is an attribute and where do we put it?

13. How do I get my link to open in a new tab?

14. What is the alt attribute used for? 
    It specifies alternative text that is to be rendered when the element to which it is applied cannot be rendered.

15. How do I reference an id?
    To reference an id, write a hash(#) character, followed by the name of the id.

16. What is the difference between a section and a div
    <section> marks up a section where content inside is grouped, <div> marks up a generic block with no associated semantics.

17. What is CSS used for? 
    CSS is used to define styles for your web pages, including the design, layout and variations in display for different devices and screen sizes.

18. How to we select an element? Example - every h2 on the page
    By using the html tags/elements. To select all h2, we can do 
    h2 {
        property: value;
    }

19. What is the difference between a class and an id? - Give me an example of when I might use each one
    An ID can be used to identify one element and name preceded by a hash character(#). It can be used in uniquely adding attributes to an element like <h1 id="main-content"></h1>
    
    A class can be used to identify more than one and preceded by a full stop(.). It can be used with mutliple assignments like all image of the page <img class="img-responsive"/>

20. How do we select classes in CSS?
    To select elements with a specific class, write a period (.) character, followed by the name of the class.

21. How do we select a p element with a single class of “human””?

22. What is a parent child selector? When would this be useful?
    It describes relationship between elements. It is useful when we are selecting descendants of an element or limit the elements found only under the parent selector.

23. How do you select all links within a div with the class of sidebar?

24. What is a pseudo selector?
    A CSS pseudo-class is a keyword added to a selector that specifies a special state of the selected element(s).
